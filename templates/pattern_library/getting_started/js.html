Creating Java Script for PROJECT NAME

Java Script is a key component of any web app and PROJECT NAME is no different. AIUI does, however, have a unique approach to using JScript. We use a jScript transcitpion tool called Brython.

Brython enables us to use python as a scripting language in the client and so maintain a single coding approach throughout the project. An example of brython code can be seen in the HB datepicker component based on the JQuery Datepicker.
