The _base.html Template

The _base.html template forms the basis for the whole site. It contains key markers (Django template {% verbatim %}{% block name %}{% endblock name %}{% endverbatim %} tags) that allow for customisation of other pages through the idea of inheritance. These blocks are points in the ideal organism where extra molecules can be inserted to realise a whole new actual organism or form the basis for a new ideal organism.

This provides structure and forms the backbone for subsequent Project Orion templates.
